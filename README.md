Telco Customer Churn — Risk and Revenue Analysis

This project analyzes telecom customer churn using a structured, data-driven approach without machine learning.

The focus is on identifying high-risk customer segments, quantifying revenue exposure, and designing financially prioritized retention strategies.

Key Highlights
	•	Overall churn rate: 26.5%
	•	Revenue at risk: 30.5%
	•	High-risk segment churn: 67.8%
	•	Extreme cluster (Month-to-month + Fiber optic) shows 54.6% churn
	•	20% churn reduction can recover ~564K annually
	•	Break-even retention cost per targeted customer: 151 per year

What This Project Demonstrates
	•	Multi-dimensional churn profiling
	•	Rule-based risk scoring framework
	•	Revenue-weighted prioritization
	•	Scenario and break-even modeling
